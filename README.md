#สารคดีกำเนิดสิ่งมีชีวิต<img width="400" height="400" alt="1000005601" src="https://github.com/user-attachments/assets/b0336df8-3230-4906-b939-64b4f10f53f6" />
สารคดีกำเนิดมนุษย์ของฉัน(ไม่อนุญาติให้คัดลอก)

สิ่งที่น่าจะเป็นไปได้มากที่สุด คนไม่ได้วิวัฒนาการมาจากสัตว์ชนิดใดและไม่สามารถกลายพันธุ์เองได้ แต่ต้นกำเนิดทุกสิ่งเกิดมาจากโลกและดาวต่างๆที่หมุนหรือเสียดสีกันไปมาจนก่อให้เกิดความร้อนหรืออากาศและก๊าซต่างๆจนทำให้มีน้ำพอเวลาผ่านไปอากาศและก๊าซที่เกิดจากการหมุนหรือใดๆก็ตามของโลกทำให้น้ำแห้งและเกิดละอองสลับหมุนเวียนไปมาทำให้ในบางพื้นผิวโลกมีน้ำขังและมีแร่ธาตุต่างๆในน้ำพอน้ำแห้ง จึงทำให้มีพืชบางชนิดที่อยู่รอดได้ ตามการหมุนเวียนของโลกซึ่งมีน้ำเป็นสารอาหารหลักจนนำมาซึ่งการออกดอกออกผลสิ่งที่คิดว่าคนและสิ่งมีชีวิตบนโลกเกิดมาจากต้นไม้คือ การออกผลของธรรมชาติและที่ทำให้ต้นกำเนิดจริงๆเราหายไปก็คือสัญชาตญาณของสิ่งมีชีวิตที่ต้องแย่งสารอาหารกันเพื่อความอยู่รอดจนทำให้บางอย่างถูกนำมาเป็นอาหารจนทำให้เกิดการขาดแคลนอาหารบนโลกนั้นอาจเป็นสาเหตุที่ทำให้มนุษย์บางกลุ่มต้องทำลายเผ่าพันธุ์ตัวเองเพราะกลัวจะมีจำนวนมากเกินไป จนทำให้สิ่งมีชีวิตชนิดอื่นไม่เพียงพอ ต่อการนำมาเป็นอาหาร คนยุคแรกจะมีอายุมากจนอาจทำให้คิดว่าตัวเองจะคงอยู่ตลอดไปพอเริ่มมีอาการหรือการแก่หรือป่วยคนยุคนั้นอาจคิดว่ามีบางอย่างเริ่มผิดปกติและเกิดการสูญพันธุ์ไม่ต่างจากสิ่งมีชีวิตชนิดอื่นที่นำมาเป็นอาหารจึงเกิดการผสมพันธุ์ของมนุษย์จนทำให้เกิดมนุษย์ขึ้นมาหลายคนและนำมาสู่ความแตกแยกแบ่งพวกแบ่งพื้นที่โดยสัญชาตญาณของสิ่งมีชีวิตจะไม่กินสิ่งมีชีวิตชนิดเดียวกันมนุษย์แข็งแรงและคล่องตัวและสามารถสื่อสารกันได้ดีกว่าสิ่งมีชีวิตชนิดอื่น นี่อาจเป็นสาเหตุที่ทำให้สิ่งมีชีวิตชนิดอื่นถูกนำมาเป็นอาหารของมนุษย์ ในการแบ่งแยกอาจจะเป็นสาเหตุให้แต่ละกลุ่มคิดหาวิธีสื่อสารกันแตกต่างออกไป หรือสิ่งที่เราเรียกกันว่าภาษาหรือการสื่อสารในรูปแบบอื่นๆจึงนำมาสู่การ ใช้เรียกสิ่งต่างๆเพื่อสื่อสารกันเฉพาะกลุ่ม
